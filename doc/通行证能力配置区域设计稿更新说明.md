# 通行证能力配置区域设计稿更新说明

更新时间：2026-07-08

## 1. 当前结论

本次设计稿将「通行证能力」收敛为账号中心下的二级配置模块，只承载账号登录、绑定手机、绑定邮箱等基础通行证能力开关，不再承担全局项目信息展示。

本地高保真稿见：

- `prototype/passport-capability/index.html`

## 2. 区域结构

| 区域 | 设计要求 |
| --- | --- |
| 位置 | 内容区顶部，位于左侧二级菜单右侧 |
| 标题 | 蓝色竖向强调条 + `通行证能力` + info icon + 辅助说明 |
| 辅助说明 | `从顶部下沉到账户中心内部，按能力卡片配置` |
| 右侧操作 | 最新版设计稿不展示 `恢复默认`，标题区右侧保持留白 |
| 标题区间距 | 标题区下方保留 24px 到卡片区 |

## 3. 能力卡片

| 卡片 | 能力说明 | 配置项 | 特殊规则 |
| --- | --- | --- | --- |
| 账密登录 | 允许用户使用账号密码登录账号中心 | 账密登录 | 默认开启；作为基础能力展示 `必选` 标签，开关禁用 |
| 绑定手机 | 在同一张卡片内配置登录手机和安全手机 | 登录手机、安全手机 | 两个配置项独立开关 |
| 绑定邮箱 | 当前仅支持配置安全邮箱 | 安全邮箱 | 不展示登录邮箱配置项 |

卡片采用一行三列等宽布局，图标、标题、说明、配置项分区固定，配置项开关统一右对齐。

## 4. 样式标注

| 元素 | 规范 |
| --- | --- |
| 标题字号 | 16px / 600 / `#1F2937` |
| 强调条 | 4px × 18px，圆角 2px，蓝色 |

| 卡片 | 白底，8px 圆角，`1px solid #E5E7EB`，24px 内边距 |
| 卡片 Hover | 边框 `#BFD7FF`，轻微蓝色阴影 |

| 开关开启 | `#2F7BFF` |
| 开关关闭 | `#D1D5DB` |
| 开关禁用 | 背景 `#E5E7EB`，滑块 `#F9FAFB`，文案 `#9CA3AF` |

## 5. 交互规则

| 交互 | 规则 |
| --- | --- |
| 开关切换 | 点击后只改变当前页面状态，不立即保存 |
| 保存 | 页面右上角 `保存` 统一提交；保存中展示 loading 文案；成功 toast 提示 `保存成功` |
| 恢复默认 | 最新版设计稿不提供该入口，本轮设计稿和原型均不展示 |
| 信息提示 | 标题旁 info icon hover 展示通行证能力说明 |

## 6. 文案口径

| 场景 | 文案 |
| --- | --- |
| 页面标题 | 通行证能力 |
| 账密登录说明 | 允许用户使用账号密码登录账号中心 |
| 绑定手机说明 | 在同一张卡片内配置登录手机和安全手机 |
| 绑定邮箱说明 | 当前仅支持配置安全邮箱 |
| 设计说明 | 该模块承接顶部移除的能力标签；顶部主体信息区仅保留基础识别信息。 |

## 7. 验收点

| 检查项 | 标准 |
| --- | --- |
| 区域职责 | 不展示全局项目信息，只展示账号中心能力开关 |
| 卡片结构 | 一行三列，顶部信息对齐，配置项底部对齐 |
| 开关 | 账密登录禁用开启；登录手机、安全手机、安全邮箱可独立切换 |
| 恢复默认 | 不展示该按钮 |
| 视觉质量 | 无异常换行、遮挡、跨列、按钮挤压 |
